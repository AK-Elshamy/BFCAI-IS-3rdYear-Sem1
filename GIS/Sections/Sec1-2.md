## Difference between google maps and GIS maps

| _Google maps_                                           | _GIS maps_                                                                                                                               |
| ------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| - define location (source,destination)<br>- create path | - define location (source,destination)<br>- create path<br>- combine and analyze data<br>- visualization of data (like traffic info)<br> |

# Map layers

GIS uses layers to lay out the data like :

- streets
- land use
- elevation
- imagery
- etc...
  which is useful for processing different types of information

## IS VS GIS

| _IS_                                                                                            | _GIS_                                                                                                                     |
| ----------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| a system that processes data using a database and other main features of any information system | the same like IS BUT has geographical data visualization and more geographic features that help represent the data better |
